Bienvenue sur CatWeatheme

Nous avons travaillés sur une extension de navigateur Chrome, le but de cette extension est de donner sa localisation (ou une autre) et que par la suite le temps nous est donné.
Si l'utilisateur choisit d'activer le thème musique, une playlist spotify lié à la météo sera lancé, et si l'utilisateur choisit d'activer le thème image, une image random lié à la météo est posté sur l'interface chrome.

On utilise 3 API sur l'extension, une pour la météo, spotify, et pour les photos.
L'extension a été fait en JS, CSS, HTML

Ce projet a été réalisé par Charlène, Marie et Anaël
